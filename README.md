
# Merlin's backend API

Setting up a simple express api server.

## Setup

```
npm i

# run the conversion scaffold
node conversation.js

# run the REST server
node app.js
```


## API endpoint

POST /talk

POST body:
```
{
    'message': 'hello!'
}
```

POST reply:
```
{
    'source': 'hello!',
    'response': 'hello fellow wizard!'
}
```
